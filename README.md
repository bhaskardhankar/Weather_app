# Weather App 🌤️

A simple and responsive weather app built with HTML, CSS, and JavaScript that shows real-time weather data for any city in the world.

## Features
- Search weather by city name
- Displays temperature, humidity, and wind speed
- Dynamic weather icons based on conditions (Rain, Clear, Clouds, Snow, Mist, Drizzle)
- Built using OpenWeatherMap API

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- OpenWeatherMap API

## How to Use
1. Clone the repository
   git clone https://github.com/bhaskardhankar/Weather_app.git
2. Open index.html in your browser
3. Type any city name and press Search or Enter

## API
This app uses the [OpenWeatherMap API](https://openweathermap.org/api).  
To use your own key, replace the `apiKey` value in `index.html`.
